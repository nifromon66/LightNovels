[[Levels]] are a way to quantify the power and experience of an entity. it's format is : "Node Rank" - "[[Levels]]".

| Node         | Level     |
| ------------ | --------- |
| No Node      | 0 - 99    |
| First Node   | 100 - 199 |
| Second Node  | 200 - 299 |
| Third Node   | 300 - 399 |
| Fourth Node  | 400 - 499 |
| Fifth Node   | 500 - 599 |
| Sixth Node   | 600 - 699 |
| Seventh Node | 700 - 799 |
| Eighth Node  | 800 - 899 |
| Ninth Node   | 900 - 999 |
